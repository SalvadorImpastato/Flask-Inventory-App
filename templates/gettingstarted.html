Getting Started with Inventory Manager

Welcome to Inventory Manager, app that allows you to manage and search for products using a simple API or web interface. This page will guide you through the basics of using the app and its API.

Using the App

The Inventory Manager app provides several HTML pages that allow you to interact with the app. Here are the available pages and their functions:

  • Manage Products: This page allows you to add, update, and delete products in the app's database. On this page can utilize the endpoint /products with the convience of the website.
  • Update Product: This page allows you to update the details of existing products in the app’s database. On this page can utilize the endpoint updaateproduct with the convience of the website.
  • List Products: This page displays a list of all products in the app's database. On this page can utilize the endpoint /listproducts with the convience of the website.
  • Search Products: This page allows you to search for products using Elasticsearch. On this page can utilize the endpoint /search with the convience of the website.
  • Product Analytics: This page displays aggregated product data, such as the total count of products, the most popular product category, and the average price of products. On this page can utilize the endpoint /analytics with the convience of the website.

Using the API

The Inventory Manager app provides an API that allows you to interact with the app programmatically. The API uses standard HTTP methods (GET, POST, PUT, DELETE) to perform operations on your product data. Here are the API endpoints and their functions:

Make sure to replace http://localhost:5000/products with the correct URL for you API's enpoints! For example http://Hosting URL:correct port/prodcuts

  • Add Products: You can add a new product to the app's database by sending a POST request to the /products endpoint with a JSON payload that contains the product data.
    • Using PowerShell: 
    Invoke-WebRequest -Method POST -ContentType "application/json" -Body '{"ProductName": "Mystical Unicorn Plushie", "ProductCategory": "Toys", "Price": 14.99, "AvailableQuantity": 20}' -Uri http://localhost:5000/products
    Using cURL: 
    curl -X POST -H "Content-Type: application/json" -d '{"ProductName": "Mystical Unicorn Plushie", "ProductCategory": "Toys", "Price": 14.99, "AvailableQuantity": 20}' http://localhost:5000/products
  • Get Products: You can retrieve a list of all products in the app's database by sending a GET request to the /products endpoint.:
    • Using PowerShell: 
    Invoke-WebRequest -Method GET -Uri http://localhost:5000/products | Select-Object -ExpandProperty Content
    Using cURL: 
    curl http://localhost:5000/products
  • Fetch a Specific Product: You can retrieve a specific product from the app's database by sending a GET request to the /products/{id} endpoint, where {id} is the ID of the product you want to fetch.
    • Using PowerShell: 
    Invoke-WebRequest -Method GET -Uri http://localhost:5000/products/{id} | Select-Object -ExpandProperty Content
    Using cURL: 
    curl http://localhost:5000/products/{id}
  • Update an Existing Product: You can update an existing product in the app's database by sending a PUT request to the /products/{id} endpoint, where {id} is the ID of the product you want to update.
    • Using PowerShell: 
    Invoke-WebRequest -Method PUT -ContentType "application/json" -Body '{"ProductName": "Mystical Plushie", "ProductCategory": "Toys", "Price": 14.99, "AvailableQuantity": 20}' -Uri http://localhost:5000/products/{id}
    Using cURL: 
    curl -X PUT -H "Content-Type: application/json" -d '{"ProductName": "Mystical Plushie", "ProductCategory": "Toys", "Price": 14.99, "AvailableQuantity": 20}' http://localhost:5000/products/{id}
  • Delete a Product: You can delete an existing product from the app's database by sending a DELETE request to the /products/{id} endpoint, where {id} is the ID of the product you want to delete.
    • Using PowerShell: 
    Invoke-WebRequest -Method DELETE -Uri http://localhost:5000/products/{id}
    Using cURL: 
    curl -X DELETE http://localhost:5000/products/{id}
  • Elasticsearch:You can search for products using Elasticsearch by sending a GET request to `http://localhost:5000/products/search?query={query}` where `{query}` is your search query.
    • Using PowerShell: 
    Invoke-WebRequest -Method GET -Uri http://localhost:5000/products/search?query=Plushie | Select-Object -ExpandProperty Content
    Using cURL: 
    curl http://localhost:5000/products/search?query=Plushie
  • Analytics:You can get aggregated product data such as total count of products, most popular product category and average price of products by sending a GET request to `http://localhost:5000/products/analytics`.
    • Using PowerShell: 
    
    Invoke-WebRequest -Method GET -Uri http://localhost:5000/products/analytics | Select-Object -ExpandProperty Content
    Using cURL: 
    curl http://localhost:5000/products/analytics

Please note that you will need to replace {id} with the actual ID of the product you want to fetch, update, or delete.
